## 💡專案使用說明

1. 使用git複製本專案
```
git clone https://github.com/Berutorion/titan_shop_front_end.git
```

2. 下載完畢後，進到專案資料夾
```
cd titan_shop_front_end
```

3. 操作以下指令安裝所需的套件
```
npm install
```

4. 在專案根目錄執行以下指令，在本地啟動專案
```
npm run start
```
5. 網頁成功連接後可使用`Local`或者`Network`的網址資訊在瀏覽器中開啟


6. 開發完成後，於專案根目錄下使用此指令打包專案用於佈署
```
./deploy.sh
```
